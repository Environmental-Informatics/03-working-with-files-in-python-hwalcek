Evaluate_Racoon_Life.py

This program 
1 - opens the file 2008Male00006.txt
2 - stores data from the file into a dictionary
3 - changes the numbers in the columns into floats
4 - writes functions to calculate the sum of a list, mean of a list, and distance between two lists
5 - uses these functions to calculate movement, average energy level, average X, average Y, and total distance moved
6 - creates output file Georges_life.txt which writes new variables in a header and displays date, time, X, Y, asleep flag, behavior mode, and distance traveled in columns

input (required) - 2008Male00006.txt
output (created) - Georges_life.txt
